# Squad Audit — Team Analysis & Chemistry Tools

Team analysis tools to audit squad chemistry, roles, and dynamics. Free squad audit for sports teams, gaming clans, and work groups.

**Live site: [https://squad.csskey.com](https://squad.csskey.com)**

## Features

- **Squad chemistry score** — Rate how well your team fits
- **Role analysis** — Identify roles and gaps
- **Personality fit** — Match teammates by traits
- **Strengths & weaknesses** — Team-level audit
- **Shareable report** — Export your squad audit

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **SEO:** react-helmet-async, JSON-LD Schema
- **Hosting:** Cloudflare Workers

## Quick Start

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```
